# Homebrew Tap for MacRelay

Open-source MCP server that gives AI assistants access to native macOS apps. Local, privacy-first, no cloud.

## Install

```bash
brew install --cask drbarq/tap/macrelay
```

This installs **MacRelay.app** to `/Applications` with:
- Menu bar manager for toggling services and permissions
- MCP server (71 tools across 13 services)
- Claude Desktop extension with icon

## Getting Started

1. **Open MacRelay** from Applications or Spotlight
2. **Enable in Claude Desktop** — Settings > Connectors > MacRelay > toggle on
3. **Start chatting** — try *"What's on my calendar this week?"*

## Uninstall

```bash
brew uninstall --cask macrelay
```

Or click **Uninstall MacRelay...** from the menu bar dropdown.
